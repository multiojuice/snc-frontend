Shapes and Colors

description

Our project aims to demonstrate how computational geometry can be applied to an artistic application. We will accept images as input from users and use geometric concepts to turn them into stained glass or mosaic-esque versions. The original inspiration for this project was using voronoi diagrams to create the stained glass window look, but we added Sobel edge detection and k-means clustering to improve the results as well. The edge detection helps to maintain the original edge boundaries of the image, and the k-means works to create a reduced color palette which is applied to the resulting image.

process overview

  1. User inputs image
  2. Run k-means clustering to recolor the image with a reduced color palette, store for later
  3. Run Sobel edge detection
  4. Utilize a gaussian blur and image subtraction to set up a probability distribution which lets us sample voronoi points that are near (but not on) edges
  5. Sample some number of points and create a voronoi diagram with this point set
  6. Color each voronoi region using the result of the k-means recoloring
  7. Return the image to the user
